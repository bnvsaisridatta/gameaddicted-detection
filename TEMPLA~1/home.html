GAMEADDICTED AMachine Learning Framework for Digital Game Addiction Detection and Early Intervention

Project Abstract

Digital Game Addiction (DGA) has emerged as a significant behavioral issue due to the rapid growth of online gaming platforms and increased digital engagement. Early detection of addictive gaming behavior is essential to enable timely intervention and reduce negative psychological and social impacts. Previous research proposed a machine learning framework for detecting digital game addiction using textual data collected from online gaming platforms and evaluated several classification algorithms including Logistic Regression, K-Nearest Neighbors, Support Vector Machines, Decision Trees, Random Forest, and XGBoost, where XGBoost achieved the highest accuracy. However, these approaches relied on individual machine learning models, which may limit predictive performance when handling complex textual patterns. This study extends the existing framework by introducing an advanced ensemble learning approach using a Stacking Classifier for improved digital game addiction detection. The proposed system integrates multiple base learners, including Random Forest, Gradient Boosting, and Support Vector Machines, while Logistic Regression is used as the meta-learner to combine their predictions. Textual data undergoes preprocessing and feature extraction using Natural Language Processing techniques and TF-IDF vectorization before being used for model training. The stacking-based architecture leverages the strengths of multiple classifiers to improve generalization and prediction accuracy. Experimental results demonstrate that the proposed stacking ensemble model significantly improves classification performance, achieving an accuracy of 98.30% with strong precision, recall, and F1-score across all classes. The results indicate that ensemble-based meta-learning techniques provide superior performance compared to single-model approaches for digital addiction detection. The proposed system contributes an enhanced predictive framework that can support automated detection and early awareness systems for identifying digital game addiction through textual analysis.
